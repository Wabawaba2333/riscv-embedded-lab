# RISC-V Embedded Systems Project

## Description
This project is an embedded systems project developed on a RISC-V microcontroller.
The firmware is written in C and Assembly and uploaded to the board using DFU.
The system includes multiple peripherals such as LCD, keypad, GPIO, UART, ADC,
and an 8x8 LED matrix.

## Features
- GPIO control
- LCD display control
- Keypad input
- 8x8 LED matrix game
- UART communication
- ADC input
- Timer and interrupt handling
- Written in C and Assembly
- Built with Makefile and uploaded via DFU

## Hardware
- RISC-V MCU board
- LCD display
- Keypad
- 8x8 LED matrix
- GPIO interfaces

## What I learned
- Microcontroller programming
- GPIO and peripheral control
- Interrupts and timers
- LCD and LED matrix interfacing
- Keypad input handling
- Low-level programming in C and Assembly
- Embedded system debugging
